# Hut jumping

> You climb up on an unlit lamppost, scanning the surroundings. You hear footsteps nearby, so you hurry over to the adjoining rooftop with a small jump. A tile comes loose and falls to the ground, causing a clatter. You'll have to be more careful next time.

The **Hut jumping**  [activity](index.md) is a  [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Coldington](https://wiki.walkscape.app/wiki/Special:MyLanguage/Coldington "Special:MyLanguage/Coldington") in  [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirement

Skill Level Requirements:

- At least  Agility lvl. 25.
