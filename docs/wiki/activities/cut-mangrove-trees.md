# Cut mangrove trees

> While being careful not to trip over their entangling roots, you are working hard to get some logs of these fascinatingly shaped trees.

The **Cut mangrove trees** [activity](index.md) is a [Woodcutting](../skills/woodcutting.md) activity.

# Location

This activity can be done at the following location:

- [Mangrove Forest](https://wiki.walkscape.app/wiki/Special:MyLanguage/Mangrove_Forest "Special:MyLanguage/Mangrove Forest") in [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Woodcutting lvl. 50.

Keyword Requirements:

- Have [Hatchet](https://wiki.walkscape.app/wiki/Special:MyLanguage/Hatchet_Keyword "Special:MyLanguage/Hatchet Keyword") equipped.
