# Cut bamboo

> You strike the bamboo with your hatchet. It makes a satisfying crunching sound.

The **Cut bamboo** [activity](index.md) is a [Woodcutting](../skills/woodcutting.md) activity.

# Location

This activity can be done at the following location:

- [Witched Woods](https://wiki.walkscape.app/wiki/Special:MyLanguage/Witched_Woods "Special:MyLanguage/Witched Woods") in [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Woodcutting lvl. 45.

Keyword Requirements:

- Have [Hatchet](https://wiki.walkscape.app/wiki/Special:MyLanguage/Hatchet_Keyword "Special:MyLanguage/Hatchet Keyword") equipped.
