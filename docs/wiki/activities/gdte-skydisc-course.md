# GDTE skydisc course

> In intermediate skydisc golfing, you're developing a keen sense of strategy. Each course presents a new challenge, as you adeptly curve your disc around obstacles, honing your skills with every calculated throw.

The **GDTE skydisc course**  [activity](index.md) is a  [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Bilgemont Port](https://wiki.walkscape.app/wiki/Special:MyLanguage/Bilgemont_Port "Special:MyLanguage/Bilgemont Port") in  [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

This activity is always visible.

Skill Level Requirements:

Keyword Requirements:
