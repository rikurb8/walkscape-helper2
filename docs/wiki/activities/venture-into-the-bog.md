# Venture into the bog

> You venture into the gloomy mists outside of Granfiddich, trying to find the landmarks you recognize from following the bats, to get into the bog.

The **Venture into the bog** [activity](index.md) is a [Agility](../skills/agility.md) activity. This is a [movement activity](https://wiki.walkscape.app/wiki/Special:MyLanguage/Movement_Activities "Special:MyLanguage/Movement Activities"), you will be teleported to [Bog Top](https://wiki.walkscape.app/wiki/Special:MyLanguage/Bog_Top "Special:MyLanguage/Bog Top") upon completion of an action.

# Location

This activity can be done at the following location:

-   [Granfiddich](https://wiki.walkscape.app/wiki/Special:MyLanguage/Granfiddich "Special:MyLanguage/Granfiddich") in [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This is a **hidden** activity. It is only visible if you have done the following:

-   Complete actions of the [Spring bat tracking](spring-bat-tracking.md) activity. **\[1\]**

# Requirements

Skill Level Requirements:

-   At least Agility lvl. 50.

Keyword Requirements:

-   Requires **\[2\]** unique [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
