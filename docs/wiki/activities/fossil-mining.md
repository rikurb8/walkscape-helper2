# Fossil mining

> Lured by the wonders of the past you focus your attention on finding ancient fossils.

The **Fossil mining**  [activity](index.md) is a  [Mining](../skills/mining.md) activity.

# Location

This activity can be done at the following location:

- [Darktide Trench](https://wiki.walkscape.app/wiki/Special:MyLanguage/Darktide_Trench "Special:MyLanguage/Darktide Trench") in  [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Mining lvl. 35.

Keyword Requirements:

- Have  [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
- Have  [Pickaxe](https://wiki.walkscape.app/wiki/Special:MyLanguage/Pickaxe_Keyword "Special:MyLanguage/Pickaxe Keyword") equipped.
- Requires **[3]** unique  [Expert diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Expert_diving_gear_Keyword "Special:MyLanguage/Expert diving gear Keyword") equipped.
