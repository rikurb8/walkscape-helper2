# Rescue team

> Use information from the search teams to actively go and carry out rescue operations.

The **Rescue team**  [activity](index.md) is a  [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Horn of Respite](https://wiki.walkscape.app/wiki/Special:MyLanguage/Horn_of_Respite "Special:MyLanguage/Horn of Respite") in  [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Agility lvl. 30.

Keyword Requirements:

- Have  [Climbing gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Climbing_gear_Keyword "Special:MyLanguage/Climbing gear Keyword") equipped.
- Have  [Skis](https://wiki.walkscape.app/wiki/Special:MyLanguage/Skis_Keyword "Special:MyLanguage/Skis Keyword") equipped.

Other Requirements:

- Have  **[20]** achievement points.
