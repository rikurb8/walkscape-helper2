# Spring bat tracking

> The bats leave the cave flying towards the north-east. Following the bats from their spring home, running day and night to find where their next home is. You might not be back here for a while.

The **Spring bat tracking**  [activity](index.md) is a  [Agility](../skills/agility.md) activity.
This is a [movement activity](https://wiki.walkscape.app/wiki/Special:MyLanguage/Movement_Activities "Special:MyLanguage/Movement Activities"), you will be teleported to  [Bog Top](https://wiki.walkscape.app/wiki/Special:MyLanguage/Bog_Top "Special:MyLanguage/Bog Top") upon completion of an action.

# Location

This activity can be done at the following location:

- [Winter's End](https://wiki.walkscape.app/wiki/Special:MyLanguage/Winter%27s_End "Special:MyLanguage/Winter's End") in  [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This is a **hidden** activity.
It is only visible if you have done the following:

- Complete actions of the  [Spring cave foraging](spring-cave-foraging.md) activity. **[10]**

# Requirements

Skill Level Requirements:

- At least  Agility lvl. 35.

Keyword Requirements:

- Requires **[2]** unique  [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
