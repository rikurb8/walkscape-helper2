# Mine tarsilium ore (Underwater)

> The light is bad underwater, but it's not different than a cave. With tarsilium, it's always hard to see the black ore in a dark environment.

The **Mine tarsilium ore (Underwater)** [activity](index.md) is a [Mining](../skills/mining.md) activity.

# Location

This activity can be done at the following location:

- [Underwater Cave](https://wiki.walkscape.app/wiki/Special:MyLanguage/Underwater_Cave "Special:MyLanguage/Underwater Cave") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Mining lvl. 40.

Keyword Requirements:

- Have [Pickaxe](https://wiki.walkscape.app/wiki/Special:MyLanguage/Pickaxe_Keyword "Special:MyLanguage/Pickaxe Keyword") equipped.
- Requires **\[3\]** unique [Advanced diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Advanced_diving_gear_Keyword "Special:MyLanguage/Advanced diving gear Keyword") equipped.
