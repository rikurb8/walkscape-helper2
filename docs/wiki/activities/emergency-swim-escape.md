# Emergency swim escape

> You've lost your diving gear and are running out of air. You must swim to the shore as fast as you can, but without panicking. You know that if you ascend too quickly, you could suffer from decompression sickness.

The **Emergency swim escape** [activity](index.md) is a [Agility](../skills/agility.md) activity. This is a [movement activity](https://wiki.walkscape.app/wiki/Special:MyLanguage/Movement_Activities "Special:MyLanguage/Movement Activities"), you will be teleported to [Port Skildar](https://wiki.walkscape.app/wiki/Special:MyLanguage/Port_Skildar "Special:MyLanguage/Port Skildar") upon completion of an action.

# Locations

This activity can be done at the following locations:

-   [Casbrant's Grave](https://wiki.walkscape.app/wiki/Special:MyLanguage/Casbrant%27s_Grave "Special:MyLanguage/Casbrant's Grave") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")
-   [Darktide Trench](https://wiki.walkscape.app/wiki/Special:MyLanguage/Darktide_Trench "Special:MyLanguage/Darktide Trench") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")
-   [Elara's Lagoon](https://wiki.walkscape.app/wiki/Special:MyLanguage/Elara%27s_Lagoon "Special:MyLanguage/Elara's Lagoon") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")
-   [Kelp Forest](https://wiki.walkscape.app/wiki/Special:MyLanguage/Kelp_Forest "Special:MyLanguage/Kelp Forest") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")
-   [Underwater Cave](https://wiki.walkscape.app/wiki/Special:MyLanguage/Underwater_Cave "Special:MyLanguage/Underwater Cave") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")
-   [Vastalume](https://wiki.walkscape.app/wiki/Special:MyLanguage/Vastalume "Special:MyLanguage/Vastalume") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This is an **emergency** activity. It is only visible if you _don't_ meet the following keyword requirements:

-   Requires **\[3\]** unique [Diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Diving_gear_Keyword "Special:MyLanguage/Diving gear Keyword") equipped.

# Requirement

Skill Level Requirements:

-   At least Agility lvl. 1.
