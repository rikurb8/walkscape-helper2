# Gold panning

> This is rudimentary gold mining technique that takes advantage of sedimentary deposits in rivers. First, the pan is submerged in the bottom sand. Then, by shaking it by hand, the gravel is loosened until only the sand and possible gold nuggets are left.

The **Gold panning** [activity](index.md) is a [Mining](../skills/mining.md) activity.

# Location

This activity can be done at the following location:

- [Norsack Plains](https://wiki.walkscape.app/wiki/Special:MyLanguage/Norsack_Plains "Special:MyLanguage/Norsack Plains") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Mining lvl. 15.
- At least Fishing lvl. 5.

Keyword Requirements:

- Have [Gold pan](https://wiki.walkscape.app/wiki/Special:MyLanguage/Gold_pan_Keyword "Special:MyLanguage/Gold pan Keyword") equipped.
