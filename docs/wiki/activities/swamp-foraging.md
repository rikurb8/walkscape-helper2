# Swamp foraging

> You search through the murky land and water for valuable flowers and plants.

The **Swamp foraging**  [activity](index.md) is a  [Foraging](../skills/foraging.md) activity.

# Location

This activity can be done at the following location:

- [Bog Top](https://wiki.walkscape.app/wiki/Special:MyLanguage/Bog_Top "Special:MyLanguage/Bog Top") in  [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Foraging lvl. 45.

Keyword Requirements:

- Requires **[2]** unique  [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
