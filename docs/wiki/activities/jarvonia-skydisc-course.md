# Jarvonia skydisc course

> As a beginner in skydisc golfing, each throw is a new discovery. You learn to gauge the wind and navigate around trees, enjoying the game's simplicity and the joy of watching the disc soar towards the basket.

The **Jarvonia skydisc course** [activity](index.md) is a [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Barbantok](https://wiki.walkscape.app/wiki/Special:MyLanguage/Barbantok "Special:MyLanguage/Barbantok") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Agility lvl. 10.

Keyword Requirements:

- Have [Skydisc](https://wiki.walkscape.app/wiki/Special:MyLanguage/Skydisc_Keyword "Special:MyLanguage/Skydisc Keyword") equipped.
