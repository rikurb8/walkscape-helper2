# Kelp foraging

> With an abundance of kelp swaying around you, you meticulously inspect each one before carefully plucking the best.

The **Kelp foraging** [activity](index.md) is a [Foraging](../skills/foraging.md) activity.

# Location

This activity can be done at the following location:

- [Kelp Forest](https://wiki.walkscape.app/wiki/Special:MyLanguage/Kelp_Forest "Special:MyLanguage/Kelp Forest") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Foraging lvl. 35.
- At least Woodcutting lvl. 35.

Keyword Requirements:

- Requires **\[3\]** unique [Diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Diving_gear_Keyword "Special:MyLanguage/Diving gear Keyword") equipped.
