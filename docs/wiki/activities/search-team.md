# Search team

> Help the adventuring guild locate and find information on open permits that have not returned.

The **Search team** [activity](index.md) is a [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Horn of Respite](https://wiki.walkscape.app/wiki/Special:MyLanguage/Horn_of_Respite "Special:MyLanguage/Horn of Respite") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirement

Skill Level Requirements:
