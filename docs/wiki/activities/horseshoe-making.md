# Horseshoe making

> Using strong metal, you forge a shoe that will protect a horse's hoof in medium to hard terrain. Although nails are used for its fixation, the horse does not suffer since the hoof is not sensitive, similar to a very thick human fingernail.

The **Horseshoe making** [activity](index.md) is a [Smithing](../skills/smithing.md) activity.

# Location

This activity can be done at the following location:

- [Port Skildar](https://wiki.walkscape.app/wiki/Special:MyLanguage/Port_Skildar "Special:MyLanguage/Port Skildar") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Smithing lvl. 20.

Keyword Requirements:

- Have [Smithing hammer](https://wiki.walkscape.app/wiki/Special:MyLanguage/Smithing_hammer_Keyword "Special:MyLanguage/Smithing hammer Keyword") equipped.
