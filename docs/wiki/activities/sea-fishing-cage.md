# Sea fishing (Cage)

> A nutrient dense current brings all sorts of crustaceans to the area.

The **Sea fishing (Cage)** [activity](index.md) is a [Fishing](../skills/fishing.md) activity.

# Location

This activity can be done at the following location:

- [Winter's End](https://wiki.walkscape.app/wiki/Special:MyLanguage/Winter%27s_End "Special:MyLanguage/Winter's End") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Fishing lvl. 35.

Keyword Requirements:

- Have [Fishing cage](https://wiki.walkscape.app/wiki/Special:MyLanguage/Fishing_cage_Keyword "Special:MyLanguage/Fishing cage Keyword") equipped.
