# Underwater aqueduct digging

> Dig the way for the carpenters who come behind you. Removing the debris in the way, while ensuring the area doesn't collapse.

The **Underwater aqueduct digging**  [activity](index.md) is a  [Mining](../skills/mining.md) activity.

# Location

This activity can be done at the following location:

- [Bog Bottom](https://wiki.walkscape.app/wiki/Special:MyLanguage/Bog_Bottom "Special:MyLanguage/Bog Bottom") in  [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Mining lvl. 40.

Keyword Requirements:

- Have  [Pickaxe](https://wiki.walkscape.app/wiki/Special:MyLanguage/Pickaxe_Keyword "Special:MyLanguage/Pickaxe Keyword") equipped.
- Requires **[2]** unique  [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
- Requires **[3]** unique  [Advanced diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Advanced_diving_gear_Keyword "Special:MyLanguage/Advanced diving gear Keyword") equipped.
