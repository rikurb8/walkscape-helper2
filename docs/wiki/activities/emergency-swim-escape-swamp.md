# Emergency swim escape (Swamp)

> You've lost your diving gear and are running out of air. You must swim to the shore as fast as you can, but without panicking. You know that if you ascend too quickly, you could suffer from decompression sickness.

The **Emergency swim escape (Swamp)** [activity](index.md) is a [Agility](../skills/agility.md) activity. This is a [movement activity](https://wiki.walkscape.app/wiki/Special:MyLanguage/Movement_Activities "Special:MyLanguage/Movement Activities"), you will be teleported to [Old Arena Ruins](https://wiki.walkscape.app/wiki/Special:MyLanguage/Old_Arena_Ruins "Special:MyLanguage/Old Arena Ruins") upon completion of an action.

# Location

This activity can be done at the following location:

-   [Bog Bottom](https://wiki.walkscape.app/wiki/Special:MyLanguage/Bog_Bottom "Special:MyLanguage/Bog Bottom") in [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This is an **emergency** activity. It is only visible if you _don't_ meet the following keyword requirements:

-   Requires **\[3\]** unique [Diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Diving_gear_Keyword "Special:MyLanguage/Diving gear Keyword") equipped.

# Requirement

Skill Level Requirements:

-   At least Agility lvl. 1.
