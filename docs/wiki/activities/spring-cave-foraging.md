# Spring cave foraging

> Bats make their home in the springtime here. It's dark, better bring a light.

The **Spring cave foraging** [activity](index.md) is a [Foraging](../skills/foraging.md) activity.

# Location

This activity can be done at the following location:

- [Winter's End](https://wiki.walkscape.app/wiki/Special:MyLanguage/Winter%27s_End "Special:MyLanguage/Winter's End") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Foraging lvl. 35.

Keyword Requirements:

- Have [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
