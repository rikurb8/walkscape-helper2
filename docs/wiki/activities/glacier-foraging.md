# Glacier foraging

> You search far and wide through the empty glacier. You learn a lot about yourself, but is there anything here to find?

The **Glacier foraging** [activity](index.md) is a [Foraging](../skills/foraging.md) activity.

# Location

This activity can be done at the following location:

- [Winter Waves Glacier](https://wiki.walkscape.app/wiki/Special:MyLanguage/Winter_Waves_Glacier "Special:MyLanguage/Winter Waves Glacier") in [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Foraging lvl. 30.
- At least Agility lvl. 30.
