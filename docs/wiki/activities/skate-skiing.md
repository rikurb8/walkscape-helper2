# Skate skiing

> Skate skiing brings a sense of adventure. You tackle varied terrains with newfound confidence, each stride stronger and more assured, as you explore the vast, snow-covered expanses.

The **Skate skiing**  [activity](index.md) is a  [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Noiseless Pass](https://wiki.walkscape.app/wiki/Special:MyLanguage/Noiseless_Pass "Special:MyLanguage/Noiseless Pass") in  [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Agility lvl. 20.

Keyword Requirements:

- Have  [Skis](https://wiki.walkscape.app/wiki/Special:MyLanguage/Skis_Keyword "Special:MyLanguage/Skis Keyword") equipped.
