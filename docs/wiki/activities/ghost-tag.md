# Ghost tag

> You chase around the spirited ghosts of Wraithwater participating in a game of tag. Each time it feels like you get close, they take the opportunity to phase through something. It feels like you will never not be 'it' against these opponents.

The **Ghost tag** [activity](index.md) is a [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Wraithwater](https://wiki.walkscape.app/wiki/Special:MyLanguage/Wraithwater "Special:MyLanguage/Wraithwater") in [Wallisia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Wallisia "Special:MyLanguage/Wallisia")

# Visibility

This activity is always visible.

# Requirement

Skill Level Requirements:

- At least Agility lvl. 55.
