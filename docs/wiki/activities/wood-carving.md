# Wood carving

> Shaping wood using a sharp object, such as a knife, is an ancient form of craftsmanship. There is a great variety of art and iconography: from small figurines and toys, to ceremonial or war masks, to large mosaics and towering totem poles.

The **Wood carving** [activity](index.md) is a [Carpentry](../skills/carpentry.md) activity.

# Location

This activity can be done at the following location:
