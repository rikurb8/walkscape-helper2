# Spelunking

> You crawl on the hard ground like a lizard. Ominous thoughts cloud your mind almost paralyzing you. You manage to push them aside for a moment and continue. Soon enough a new unexplored path unveils itself and fills you with joy.

The **Spelunking**  [activity](index.md) is a  [Mining](../skills/mining.md) activity.

# Location

This activity can be done at the following location:

- [Black Eye Peak](https://wiki.walkscape.app/wiki/Special:MyLanguage/Black_Eye_Peak "Special:MyLanguage/Black Eye Peak") in  [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Mining lvl. 50.
- At least  Agility lvl. 50.

Keyword Requirements:

- Have  [Climbing gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Climbing_gear_Keyword "Special:MyLanguage/Climbing gear Keyword") equipped.
- Have  [Light source](https://wiki.walkscape.app/wiki/Special:MyLanguage/Light_source_Keyword "Special:MyLanguage/Light source Keyword") equipped.
