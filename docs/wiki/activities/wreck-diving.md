# Wreck diving

> You dive deep into the shipwreck that is Casbrant's grave, hoping to find treasure.

The **Wreck diving** [activity](index.md) is a [Foraging](../skills/foraging.md) activity.

# Location

This activity can be done at the following location:

-   [Casbrant's Grave](https://wiki.walkscape.app/wiki/Special:MyLanguage/Casbrant%27s_Grave "Special:MyLanguage/Casbrant's Grave") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

-   At least Foraging lvl. 35.
-   At least Agility lvl. 35.

Keyword Requirements:

-   Requires **\[3\]** unique [Diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Diving_gear_Keyword "Special:MyLanguage/Diving gear Keyword") equipped.
