# Merfolk farm foraging

> Your Communal Sea Farm access grants you the freedom to forage in the endless underwater farm fields. Here, you'll find a vast array of plants waiting to be picked.

The **Merfolk farm foraging**  [activity](index.md) is a  [Foraging](../skills/foraging.md) activity.

# Location

This activity can be done at the following location:

- [Elara's Lagoon](https://wiki.walkscape.app/wiki/Special:MyLanguage/Elara%27s_Lagoon "Special:MyLanguage/Elara's Lagoon") in  [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Foraging lvl. 40.

Keyword Requirements:

- Requires **[3]** unique  [Diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Diving_gear_Keyword "Special:MyLanguage/Diving gear Keyword") equipped.

Other Requirements:

- Have **[10]**  [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia") faction reputation.
