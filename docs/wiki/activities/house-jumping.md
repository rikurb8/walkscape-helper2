# House jumping

> With feline grace you travel around the city, using any architectural feature as help: walls, roofs, balconies, streetlights. They all bend to your wishes.

The **House jumping**  [activity](index.md) is a  [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Everhaven](https://wiki.walkscape.app/wiki/Special:MyLanguage/Everhaven "Special:MyLanguage/Everhaven") in  [Grand Duchy of Trellin-Erdwise](https://wiki.walkscape.app/wiki/Special:MyLanguage/Grand_Duchy_of_Trellin-Erdwise "Special:MyLanguage/Grand Duchy of Trellin-Erdwise")

# Visibility

This activity is always visible.

# Requirement

Skill Level Requirements:

- At least  Agility lvl. 45.
