# Mine farganite ore

> With sweat and tears, you slowly extract some of the farganite despite its tremendous hardness.

The **Mine farganite ore**  [activity](index.md) is a  [Mining](../skills/mining.md) activity.

# Location

This activity can be done at the following location:

- [Black Eye Peak](https://wiki.walkscape.app/wiki/Special:MyLanguage/Black_Eye_Peak "Special:MyLanguage/Black Eye Peak") in  [Jarvonia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Jarvonia "Special:MyLanguage/Jarvonia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least  Mining lvl. 50.

Keyword Requirements:

- Have  [Pickaxe](https://wiki.walkscape.app/wiki/Special:MyLanguage/Pickaxe_Keyword "Special:MyLanguage/Pickaxe Keyword") equipped.
