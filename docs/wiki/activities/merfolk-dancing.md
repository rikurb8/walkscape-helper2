# Merfolk dancing

> Performing the merfolk dance like the underwater folk do requires a lot of skill. With each attempt, you're getting better at moving in unison with the other dancers, until you achieve a seamless flow, resembling a school of fish in their synchronized movement.

The **Merfolk dancing** [activity](index.md) is a [Agility](../skills/agility.md) activity.

# Location

This activity can be done at the following location:

- [Vastalume](https://wiki.walkscape.app/wiki/Special:MyLanguage/Vastalume "Special:MyLanguage/Vastalume") in [Syrenthia](https://wiki.walkscape.app/wiki/Special:MyLanguage/Syrenthia "Special:MyLanguage/Syrenthia")

# Visibility

This activity is always visible.

# Requirements

Skill Level Requirements:

- At least Agility lvl. 30.

Keyword Requirements:

- Requires **\[3\]** unique [Diving gear](https://wiki.walkscape.app/wiki/Special:MyLanguage/Diving_gear_Keyword "Special:MyLanguage/Diving gear Keyword") equipped.
