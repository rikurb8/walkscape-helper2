# Equipment

Equipment are all the items in the game that can be equipped to your character for [bonuses](https://wiki.walkscape.app/wiki/Special:MyLanguage/Attributes "Special:MyLanguage/Attributes") and [abilities](https://wiki.walkscape.app/wiki/Special:MyLanguage/Abilities "Special:MyLanguage/Abilities"). There are **gear** items which can be equipped to one of the twelve gear slots, or **tools** which can be equipped to the six tool slots.

#### Other Tables

If you are looking for equipment by source you might like one of the following tables:

- [Activity Equipment](https://wiki.walkscape.app/wiki/Special:MyLanguage/Activity_Equipment "Special:MyLanguage/Activity Equipment") - All the gear and tools that can drop from activities.
- [Shop Equipment](https://wiki.walkscape.app/wiki/Special:MyLanguage/Shop_Equipment "Special:MyLanguage/Shop Equipment") - All the gear and tools that can be bought from shops.
- [Upgraded Equipment](https://wiki.walkscape.app/wiki/Special:MyLanguage/Upgraded_Equipment "Special:MyLanguage/Upgraded Equipment") - All the gear and tools that can be crafted from other gear and tools.
- Other tables in progress. Ping **Bonez565** on the discord to request.
